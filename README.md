# MiniC
A homework of compiling C-- language by flex and bison.

### Support
#### Type
- int
- float
- char

#### Control
- if
- else
- while
- for

#### Operator
- Selfplus & Selpminus
- PlusAss MinusAss StarAss & DivAss
- plus minus star & div
- compare operator

#### Other
- line Comment
- block Comment
